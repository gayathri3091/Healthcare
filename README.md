# Project Title:
    Patient admissions and hospital revenue analysis.
## Short Description: 
   This excel project will show us complete analysis and visualization of the type of admissions,medical conditions,hospital's total revenue,patient's age group also their test results.
## Problem Statement/Goal:
   Generally hospitals will be generating huge amount of data which will be difficult to handle without proper analysis and visualization.The goal of this project is to analyse the 
   patterns of patient demographics,hospital revenue,medical conditions,type of admissions which helps to take data driven decisions to improve hospital's revenue and performance.
## Data Source:
   This is the healthcare dataset,downloaded from kaggle website. 
## Tools Used:
    * Microsoft Excel.
    * Excel formulas for data cleaning.
    * Pivot tables,Charts,Slicers,Dashboards for visualization.
## Insights/Visualization:
   * Hospital's total revenue and total admissions is displayed as KPI's  in cards.
   * Bar Charts specifies total revenue vs medical conditions/type of admissions & age group count vs medical conditions.
   * Line Chart specifies count of patients admitted vs total billing amount by (year and months).
   * Pie chart specifies count of patients vs insurance provider.
